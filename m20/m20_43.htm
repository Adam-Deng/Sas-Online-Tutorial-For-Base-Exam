Selecting Variables (continued)

Suppose the Empinfo.Bonuses data set contains the variables ID, Name, Office, Manager, Location, and Amount. Specify a data set option in the MERGE statement below to use only the variables ID, Name, and Amount from the data set. 

data mergdata.emppay;
   merge sales.reps(rename=(office=OfficeNumber))
         empinfo.sales
         empinfo.bonuses ;
   by id;
run;
