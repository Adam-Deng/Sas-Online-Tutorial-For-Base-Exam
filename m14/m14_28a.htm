DATA Step Processing for the SET Statement

This animation shows the last two iterations of a DATA step.

  1. The following program is submitted to SAS: 
    data finance.duejan;
   set finance.loans;
   Interest=amount*(rate/12);
run;
  2. On the third iteration of the DATA step, the SET statement reads the third observation from the input data set and writes the values to the program data vector.

  3. Then, the assignment statement executes to compute the value for Interest.

  4. At the end of the third iteration of the DATA step, the values in the program data vector are written to the new data set as the first observation. Control returns to the top of the DATA step.

  5. At the beginning of the fourth iteration, the value of _N_ is set to 4, the value of _ERROR_ is reset to 0, and the value of Interest is reset to missing. The values of the variables that were read from the SAS data set with the SET statement are retained.

  6. As the SET statement executes, the values from the fourth observation are written to the program data vector.

  7. The assignment statement executes again to compute the value for Interest for the fourth observation.

  8. At the bottom of the DATA step, the values in the program data vector are written to the data set as the fourth observation.
