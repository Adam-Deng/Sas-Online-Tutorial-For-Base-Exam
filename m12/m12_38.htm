  1. Copy the program below and paste it into the code editing window. 
    data sasuser.sales;
   infile saledata;
   input LastName $ 1-7 Month $ 9-11
         Residential 13-21
         Commercial 23-31;
run;
proc print data=sasuser.sales;
run;
