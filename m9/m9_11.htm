Setting ODS HTML Options (continued)

Add an ODS HTML statement to the partial program shown below. Store both GIF files and HTML output in the Windows directory C:\Data\Pollen\Html. Name the HTML file sources.html in lowercase as shown. 

     

data work.disper7b;
        set zone7b.pollen;
        length BarDrill $ 40;
        if pollentype='GRASS' then bardrill='HREF="grass7b.html"';
        if pollentype='WEED' then bardrill='HREF="weed7b.html"';
        if pollentype='TREE' then bardrill='HREF="tree7b.html"';
     run;
     ods listing close;
