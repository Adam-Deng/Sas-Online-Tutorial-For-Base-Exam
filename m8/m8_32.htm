Creating Interactive Graphs (continued)

Steps in Creating Interactive Graphs

You can generate the SAS/GRAPH control for ActiveX in several ways. Here we focus on how to generate the control in a SAS/GRAPH program. Let's look at creating an interactive bar chart that is embedded in an HTML page. To use the SAS/GRAPH control for ActiveX, you must submit the following:

Submit... To...
an ODS HTML statement open the HTML destination and create HTML output
a GOPTIONS statement specify the ACTIVEX device driver for generating the output
a SAS/GRAPH procedure create the graph
an ODS HTML statement close the HTML destination.


Setting ODS HTML Options

If you completed the lesson , you learned that the ODS HTML statement writes program output to HTML files. To create only HTML output, you close ODS listing output in order to conserve system resources, and then open ODS HTML output. To identify the file that contains the HTML output, you specify the BODY= option in the ODS HTML statement.

     ods listing close;
     ods html body='barchart.htm';


Specifying the ActiveX Control

To generate graphs that are ActiveX controls, you specify DEVICE=ACTIVEX in the GOPTIONS statement. The ActiveX control will appear in barchart.htm. To size the ActiveX control, you can use the XPIXELS= and YPIXELS= options.

     ods listing close;
     ods html body='barchart.htm';
     goptions device=activex xpixels=400 ypixels=300;


Creating the Graph

Now you create the graph that you want to display. Let's add a PROC GCHART step to create a three-dimensional horizontal bar chart that displays the mean arterial pressure for different categories of shock.

     ods listing close;
     ods html body='barchart.htm';
     goptions device=activex xpixels=400 ypixels=300;
     proc gchart data=clinic.heart;
        hbar3d shock / sumvar=arterial type=mean;
     run;
     quit;
You can also specify DEVICE=ACTIVEX with the GCONTOUR, GMAP, GPLOT, and G3D procedures in SAS/GRAPH.


Closing the HTML Destination

The last step in creating the interactive graph is to close the HTML destination and all the files that are associated with it. To do this, you use the ODS HTML CLOSE statement.

     ods listing close;
     ods html body='barchart.htm';
     goptions device=activex xpixels=400 ypixels=300;
     proc gchart data=clinic.heart;
        hbar3d shock / sumvar=arterial type=mean;
     run;
     quit;
     ods html close;
