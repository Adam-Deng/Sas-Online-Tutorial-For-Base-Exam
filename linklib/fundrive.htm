SAS Data Set Funddrive

The Funddrive data set records the quarterly contributions of employees toward a charity fund.


Data Set Description

Variable Type Length Description
LastName char 9 contributing employee's name
Qtr1 num 8 1st quarter's contribution
Qtr2 num 8 2nd quarter's contribution
Qtr3 num 8 3rd quarter's contribution
Qtr4 num 8 4th quarter's contribution

Sample Data
