Before You Begin

As you take this , you will be practicing your new programming skills using sample data in the actual SAS environment.

Before you begin, make sure you follow the steps below.



Set up your learning environment
The first time you take this , do the following in SAS to set up practice data:
  1. You can take lessons without creating sample data, but you won't be able to complete practices.

  2. During practices, arrange your browser and software windows so that you can see them at the same time.



 

Check your browser
  1. To get the most from this , use Internet Explorer 5.0 or higher, or Netscape Navigator 6.0 or higher. In Netscape Navigator, make sure font size is set to 18 or lower.

  2. Enable JavaScript in your browser.

  3. Enable cookies in your browser. If you disable cookies, you can't use the resume feature (see Save Your Place below)

    Are cookies enabled in your browser? Click the button below to find out.


 

Save your
place
  1. You can bookmark pages for easy access using your browser. However, be sure to start this first (see Set Up Your Learning Environment above). Then go to the bookmarked page.

  2. This automatically saves your place. To return to the last lesson page you visited, click Resume on the home page or on the Learning Path.
