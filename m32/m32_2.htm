Understanding Macro Variables

When writing SAS programs, you might find that you need to reference the same variable, data set, or text string several times in the same program. 

     title "Temporary Employees for 1999";
     data hrd.temp1999;
        set hrd.temp;
        if year(enddate)=1999;
     run;
     proc print data=hrd.temp1999;
     run;

Then, you might need to change these multiple references so that you can run your program using a new variable, data set, or text string. Scanning your program and manually changing multiple references could take a long time, especially if your program is lengthy. 

     title "Temporary Employees for 2000";
     data hrd.temp2000;
        set hrd.temp;
        if year(enddate)=2000;
     run;
     proc print data=hrd.temp2000;
     run;

Using SAS macro variables simplifies the process of editing multiple references in programs. Macro variables substitute text in programs so that you can make one change and have the change appear throughout the program. Let's see how macro variables work.
