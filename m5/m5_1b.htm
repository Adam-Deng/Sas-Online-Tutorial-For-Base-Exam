Two-Dimensional Tabular Report

You can create a two-dimensional table, such as the one below, that displays one variable in the row dimension and two variables in the column dimension. 


     proc tabulate data=clinic.admit;
        class sex;
        var height weight;
        table sex,height*min weight*min;
     run;
