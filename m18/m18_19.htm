Create HTML output with a linked table of contents using the SAS data sets Sasuser.Admit and Sasuser.Insure.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html 
    body='<my-location>myoutput.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Replace <my-location>

  3. Add specifications to the ODS HTML statement to save a contents file as mytoc.htm and a frame file as myframe.htm. Save these files in the same location as myoutput.htm.

  4. Submit the program and open the frame file, myframe.htm. Use the links in the contents file to view the procedure output in the body file.
