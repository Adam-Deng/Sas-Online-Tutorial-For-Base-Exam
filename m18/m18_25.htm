Add an option to specify a destination for the HTML files you create.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html body='myoutput.htm'
    (url='myoutput.htm')
    contents='mytoc.htm'
    (url='mytoc.htm')
    frame='myframe.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Add an option to save myoutput.htm, mytoc.htm, and myframe.htm Add a suboption to specify that no path information be used in the links that are generated to body and contents files.

  3. Submit the program and open the frame file.
