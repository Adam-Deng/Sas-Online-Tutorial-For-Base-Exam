Creating a Basic Plot

Let's start by using the GPLOT procedure to plot one variable against another within a set of coordinate axes. You specify

  • a PROC GPLOT statement to invoke the procedure and to specify which data set to use
  • a PLOT statement to specify which variables to plot.

General form, simple PROC GPLOT step:
PROC GPLOT DATA=SAS-data-set;
        PLOT vertical-variable*horizontal-variable;
RUN;

where

  • SAS-data-set identifies the data set to be used
  • vertical-variable is the variable to be plotted on the vertical axis
  • horizontal-variable is the variable to be plotted on the horizontal axis.


In the PROC GPLOT step below, the PROC GPLOT statement invokes the procedure and specifies Clinic.Totals2000 as the data set that contains the variables to be plotted. The PLOT statement specifies NewAdmit as the vertical axis variable and Month as the horizontal axis variable. 

     proc gplot data=clinic.totals2000;
        plot newadmit*month;
     run;
