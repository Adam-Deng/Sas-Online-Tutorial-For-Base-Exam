Format variable values using Sasuser.Admit.

  1. Copy the program below and paste it into the code editing window. 
    title1 'March Admissions';
footnote1 'For Patients Over 30';
proc print data=sasuser.admit label;
   var age height weight fee;
   where age>30;
   sum fee;
   label fee='Admission Fee';
run;
  2. Insert a statement to format the values of Fee with a width of ten print positions, including dollar signs, commas, and two decimal places.

  3. Submit the program and view the output.

  4. Submit a null TITLE statement and a null FOOTNOTE statement to cancel all titles and footnotes.
