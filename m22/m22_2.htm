Writing Basic DATA Steps

If you completed the lesson , you learned how to write a DATA step to create a permanent SAS data set from raw data that is stored in an external file.

     data clinic.stress;
        infile tests;
        input ID 1-4 Name $ 6-25 RestHR 27-29
              MaxHR 31-33 RecHR 35-37
              TimeMin 39-40 TimeSec 42-43
              Tolerance $ 45;
     run;
