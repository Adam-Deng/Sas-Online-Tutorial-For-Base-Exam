Expanding Your Use of Arrays (continued)

Complete the DATA step below by adding an ARRAY statement to create five new character variables named New1-New5 with a length of 16.

Description of Hrd.Courses
Variable Type Length
Group1 char 12
Group2 char 12
Group3 char 12
Group4 char 12
Group5 char 12 


  data work.newcrs;
     set hrd.courses;
     array g{5} group1-group5;
       
     do i=1 to 5;
        new{i}=g{i}||'/new';
     end;
  run;
