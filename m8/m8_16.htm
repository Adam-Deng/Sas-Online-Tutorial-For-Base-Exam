Controlling VBAR Statistics

Next, let's consider vertical bar charts. By default, only the statistic frequency is displayed for VBAR charts (and for VBAR3D charts, which are three-dimensional vertical bar charts). To display additional statistics, you can use

  • the OUTSIDE= option to display a statistic above the bars
  • the INSIDE= option to display a statistic within the bars.

General form, VBAR and VBAR3D statements for specifying additional statistics:
VBAR | VBAR3D chart-variable / <options>
                               OUTSIDE=statistic
                               INSIDE=statistic;

where statistic indicates the statistic of the chart variable to be displayed.

You can specify any valid VBAR statistic for the OUTSIDE= and INSIDE= options, and you can use these options either separately or together. If an inside statistic is wider than the bars, it might not display properly.


In the following example, the VBAR statement specifies that the statistic FREQ (frequency) be displayed inside the bars and that the statistic CFREQ (cumulative frequency) be displayed above the bars. 
        proc gchart data=clinic.admit;
           vbar sex / group=actlevel
                patternid=group outside=cfreq
                inside=freq;
        run;

In contrast, the chart below shows the default behavior for VBAR charts. Note that unlike horizontal bar charts, only one statistic is displayed by default for vertical bar charts. 
        proc gchart data=clinic.admit;
           vbar sex / group=actlevel
                patternid=group;
        run;

As with horizontal bar charts, you can label statistics on vertical bar charts. For example, the program below replaces the text for the statistic on the response (y) axis with the label that is specified in the AXIS statement.

     axis1 label=('No. of People');
     proc gchart data=clinic.admit;
        hbar sex / group=actlevel patternid=group
             outside=cfreq inside=freq raxis=axis1;
     run;
