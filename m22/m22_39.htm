Test a program without creating a data set.

  1. Copy the program below and paste it into the code editing window. Modify the DATA step to test the program without reading any observations. 
    data sasuser.sales; 
   infile saledata; 
   input LastName $ 1-7 Month 9-11 
         Residential 13-21 
         Commercial 23-31;
   Total=residential+commercial;
run;

  2. Submit the program and examine the log for errors.

  3. Modify the DATA step so that it reads all the observations, but does not create a data set.

  4. Re-submit the program and examine the log. Notice that the error is reported once the values of the variables are read and verified.

  5. Fix the syntax error in the DATA step. Re-submit the program and examine the log for errors.

  6. Edit the DATA step to name the new data set Sasuser.Sales, and add a PROC PRINT step to list the new data set. Submit the revised program and view the output.
