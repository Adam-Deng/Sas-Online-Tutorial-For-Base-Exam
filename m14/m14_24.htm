Execution Phase (continued)

  1. At the beginning of the second iteration, the value of _N_ is set to 2, and the value of _ERROR_ is reset to 0. Remember, the automatic variable _N_ keeps track of how many times the DATA step has begun to execute.

  1. As the SET statement executes, the values from the second observation are written to the program data vector.

  1. The assignment statement executes again to compute the value for Interest for the second observation.
