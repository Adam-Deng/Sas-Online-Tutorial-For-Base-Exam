Specify an output catalog and view graphs.

  1. Copy the program below and paste it into the code editing window. 
    symbol1 value=star color=red height=2
            interpol=join;     
symbol2 value=star color=blue height=2
            interpol=join;     
proc gplot data=sasuser.therapy1999; 
   plot aerclass*month walkjogrun*month / 
        overlay vaxis=0 to 300 by 75; 
   plot swim*month; 
run;
quit;

  2. Add an option to specify Sasuser.Mygraphs as the output catalog for your graphs.

  3. Submit the program and view the overlaid plot. Notice the title of the GRAPH window.

  4. In the GRAPH window, scroll down to view the next plot. Notice the title of the GRAPH window.

  5. Using the Results window, redisplay the overlaid plot.

  6. Open the Sasuser.Mygraphs catalog and display the Gplot1 entry.
