SAS Library Engines

An engine is a part of SAS that reads from or writes to files. Each engine enables SAS to access files that are in a particular format.

There are several types of engines:

  • Library engines control access to SAS data libraries and can be specified in a LIBNAME statement. Native library engines access SAS files that are created and maintained by SAS, whereas interface library engines support access to other vendors' files, such as SPSS files or database data.

  • View engines enable SAS to read SAS data views that have been created using the SQL procedure (native view) or using SAS/ACCESS software (interface view). You cannot specify a view engine in a LIBNAME statement.
For a more detailed description of library engines, see the SAS documentation. For a list of valid engines, see the SAS documentation for your operating environment.
