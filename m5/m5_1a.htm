One-Dimensional Tabular Report

You can create a one-dimensional table, such as the one below, that displays two variables in the column dimension. 


     proc tabulate data=clinic.diabstat;
        class type;
        var premium;
        table type premium;
     run;
Type Premium
I II
N N Sum
3 17 3359.15
