Apply a new style to HTML output.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html body='<my-location>mystyle.htm'; 
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
    run;    
ods html close;
ods listing;
  2. Replace <my-location>

  3. Add an option to apply the Beige style to the HTML output.

  4. Submit the program and view the results.
